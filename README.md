# Dizzy Hackers MEGA — Cyber Archive

A high-fidelity hackathon landing page built for the **FusionX x Dizzy Hackers** event at Presidency University, Bangalore.

## Tech Stack
-   **Structure**: Semantic HTML5
-   **Styling**: Tailwind CSS (CDN), Custom Cinematic CSS Variables
-   **Motion**: Canvas-based Matrix Digital Stream & GSAP (ScrollTrigger)
-   **Assets**: Iconify (SVGs), Unsplash (Dynamic High-Res Backgrounds)

## High-Performance Background
The site features a custom `initCinematicStream` function that renders a falling digital alphanumeric stream in a Canvas element. This is optimized for low CPU usage while maintaining a rich "hacker" atmosphere.

## Deployment to Vercel

### Option 1: GitHub Integration (Recommended)
1. Initialize a Git repository: `git init`
2. Add all files: `git add .`
3. Commit: `git commit -m "Initial Mission Launch"`
4. Push to a new GitHub repository.
5. Connect your GitHub repo to **[Vercel Dashboard](https://vercel.com/new)**. It will automatically detect the static file.

### Option 2: Vercel CLI
If you have Node.js installed:
1. Install CLI: `npm i -g vercel`
2. Run `vercel` in this directory and follow the prompts.

### Option 3: Manual Drag-and-Drop
Zip the file OR just drag the folder into the **[Vercel Deploy](https://vercel.com/import/project)** area.
